# Admin Panel Guide

## Features

### 1. Events Management
- Add, edit, and delete past events
- Upload event images
- Add star ratings (1-5 stars or none)
- Auto-scrolling carousel on homepage

### 2. Site Settings
- Change About page hero image
- Change party atmosphere image
- Change homepage "me" image (DJ performing)
- All changes update across the site

## How to Use

### Accessing Admin Panel
1. Go to `your-site.com/admin`
2. Enter your admin password
3. You'll see two tabs: **Events** and **Site Settings**

### Managing Events
1. Click **Events** tab
2. Click **Add Event** to create new event
3. Fill in:
   - Event Title
   - Date (e.g., "June 21, 2023")
   - Location (e.g., "Miami Beach, FL")
   - Rating (click stars or âœ• for none)
   - Upload image (under 5MB)
4. Click **Save Event**
5. To edit: Click **Edit** on any event card
6. To delete: Click **Delete** on any event card

### Changing Site Images
1. Click **Site Settings** tab
2. You'll see three image sections:
   - **About Page Hero**: Main image on /about page
   - **Party Atmosphere**: Party crowd image on /about page
   - **Homepage About Section**: DJ performing image on homepage
3. Click **Choose File** under any image
4. Select new image from your computer
5. Preview appears immediately
6. Click **Save Changes** to apply
7. Refresh the page to see changes on the site

## Image Guidelines

### Events
- Format: JPG or PNG
- Size: Under 5MB
- Orientation: Portrait (3:4 ratio works best)
- Shows 4 events at a time in carousel

### Site Images
- Format: JPG or PNG
- Size: Under 5MB
- About Hero: Portrait orientation
- Party Image: Landscape or portrait
- Me Image: Portrait showing DJ setup

## Tips

- **Logout**: Click "Logout" in top right when done
- **Session**: Stays logged in until you close browser
- **Changes**: Some changes require page refresh to see
- **Backup**: Keep original images saved locally
- **Mobile**: Admin panel works on mobile too

## Troubleshooting

**Can't login?**
- Check password is correct
- Clear browser cache and try again

**Images not uploading?**
- Check file size (must be under 5MB)
- Try compressing at tinypng.com
- Ensure file is JPG or PNG format

**Changes not showing?**
- Hard refresh page (Ctrl+F5 or Cmd+Shift+R)
- Clear browser cache
- Wait a few seconds for changes to propagate

**Events not saving?**
- Check all required fields are filled
- Ensure you have an image uploaded
- Try logging out and back in
